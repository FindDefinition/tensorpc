## Flow

You can create a simple app with the following code:

```Python 
from tensorpc.flow import mui, three, plus, appctx, mark_create_layout, flow
class App:
    @mark_create_layout
    def my_layout(self):
        nodes = [
            flow.Node(id="a", data=flow.NodeData(label="hello"))
        ]
        return flow.Flow(nodes, [])
```
